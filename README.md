1) This project used create-react-app
2) Used fetch API to get the results from the URL
3) In order to over come the CORS issue used another API by appending the given API with 'https://cors-anywhere.herokuapp.com/' as this enables cors
4) Used DevExpress Library to display the data in the tabular form and pie chart
5) Inorder to run the app move to the downloaded folder and run "npm install " and then "npm start"
6) You can also clone the source code here : 'https://github.com/nikhilgvpce/BankTransaction.git'
